About

把节点图的复杂性隐藏在镜头卡背后

DreamChord 是一个面向非技术用户的视觉小说创作平台。用户只需像写小说一样填写背景、选择角色、编写台词，系统自动将其转换为可播放的节点图和运行时场景。无需学习 Ren'Py 脚本语言，无需理解节点连线，用"章节 → 场景 → 镜头卡"的方式组织剧情，一键发布即可在浏览器中播放。

痛点：节点图太难
镜头卡写作界面，用户只需填写"谁说了什么"，系统自动生成节点和连线
痛点：没有可视化预览
编辑器右栏实时预览，每改一个字、换一个背景立刻看到播放效果
痛点：分支管理混乱
自动检测分支去向，流程图可视化所有场景和分支连线，汇合点紫色高亮
痛点：AI 辅助缺失
编辑器内置 AI 助手，支持润色、续写、生成选项、分支回应、生成剧情图
痛点：无法即时分享
一键发布到作品广场，分享链接即可在浏览器中播放

Quick Start

三步启动项目

Node.js 20+ 环境，pnpm 或 npm 均可。

01

安装依赖

克隆仓库后安装前后端依赖

git clone https://github.com/tl66666/DreamChord.git cd DreamChord && pnpm install
02

初始化数据库

配置 .env，运行 Prisma 迁移和种子

cd apps/server cp .env.example .env npx prisma migrate dev && npx prisma db seed
03

启动开发服务器

双击 start-dreamchord.bat 或手动启动前后端

cd apps/web && npm run dev cd apps/server && npm run dev
