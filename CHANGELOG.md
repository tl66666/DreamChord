## [0.2.0] - 2026-07-12

### 新增

- 章节、场景、镜头卡和剧情流程图组成的完整故事工作台。
- 对话、旁白、心理、回忆、系统提示、人物舞台动作、选择与跳转。
- 新镜头的背景与在场人物继承，以及分支独立舞台状态计算。
- 长篇正文解析、导入前预览、场景拆分和角色映射。
- 实时场景预览、完整播放器、项目发布与分享。
- 全局素材库，支持跨项目复用、重命名、替换、引用检查和删除。
- 图片内容检查、原图保留、白底去除、透明裁边、立绘/CG/背景候选与审核。
- 多对话创作 Agent，可选章节绑定、对话管理、项目上下文和分层记忆。
- Agent 白名单工具、结构化剧情提案、校验、审批、应用与版本安全撤销。
- 无 API Key 的本地助手，以及 GLM、DeepSeek、Kimi、OpenAI 兼容模型配置。
- 故事圣经、剧情健康检查、项目备份与携带素材的严格恢复。
- Windows 一键启动、环境诊断、数据库同步前备份和重复启动复用。
- 桌面、平板与移动端响应式布局。

### 改进

- 普通 Agent 对话不再强制使用剧情补丁 JSON，模型自然文本可直接成为回答。
- 没有选择章节时，意外返回的剧情补丁会被丢弃并保留文字结论。
- Agent 失败状态和可见解释消息原子写入，避免只显示用户问题。
- 兼容部分模型返回的 `patch: null`、代码块、字段别名和 JSON 前后说明。
- 自动保存串行执行，并对旧版本保存返回冲突，防止覆盖新内容。
- 项目导入重映射内部 ID、路径与素材引用，不覆盖已有项目。

### 修复

- 修复一个分支的人物状态泄漏到另一分支的问题。
- 修复重复启动时 Prisma Windows DLL 被旧后端占用导致的常见失败路径。
- 修复只读 Agent 回复显示不适用撤销操作的问题。

### 安全

- 上传图片按解码后的真实内容检查，不只信扩展名和 MIME。
- Agent 只暴露受限业务工具，不提供任意 shell、文件系统或数据库权限。
- API Key 不写入项目记录、对话正文或运行结果。
- 项目备份使用大小限制、严格 schema、内容哈希和媒体检查。
